## 0. 핵심 명제 (Thesis)

> **음향(반사/방사) 기반 내부 형상 측정은 해가 비유일(ill-posed)하다. 수위를 변화시키며 여러 경계조건에서 측정하면 유일성이 복원된다. 이 well-posed 역문제를 딥러닝(CNN → RNN → PINN)으로 해결한다.**

- 선행분야: 음향 반사측정(acoustic reflectometry), 음향 비강측정(acoustic rhinometry/pharyngometry) — "음향으로 단면적 복원"을 하지만 **비유일성**이 약점.
- 본 연구의 신규성: **물채움을 유일성 복원 장치로 도입**한 이중 모달리티. (연속방정식은 유일성 회복의 물리적 근거 중 하나.)

---

## 1. 측정 원리 (이중 채널)

정유량 `Q`(일정하고 **알려진 값**)로 채울 때, 수면 높이 `h(t)`에서:

```
dV/dt = A(h)·dh/dt = Q   →   A(h) = Q / (dh/dt)
```

- **음향 채널**: 공명 → 공기기둥 길이 → **수위 h(t)** 측정 (+ 수면 위 형상).
- **유체(연속방정식) 채널**: 알려진 Q + h(t) → **수면(잠긴) 단면적 A(h)**.
- 물이 바닥→위로 지나며 모든 높이가 한 번씩 수면이 되므로, **연속방정식만으로 전체 A(z) 복원 가능**. 음향은 수위 제공이 주임무, 수면 위 형상 역산은 교차검증.
- 노이즈 강건화: `A(h)` 를 미분(dh/dt) 대신 **적분 제약** `∫A dh = Q·t` 형태로 사용.

**⚠ 정유량 Q 기지(旣知)는 선택이 아니라 연구의 토대.** (수위-부피 정규화는 이 토대를 파괴하므로 금지 — 아래 4장 참조.)

## 2. 일반화의 논리 (미지 공동에서 왜 되는가)

- 블랙박스 "스펙트로그램 → 반지름" 회귀는 **학습분포 밖(OOD) 실제 공동에서 일반화 안 됨.**
- **연속방정식은 형상과 무관한 보편 물리** → 어떤 미지 공동에서도 성립.
- 따라서 ML 역할 재정의: ML은 **"공명 → 수위/공기기둥 길이"** 라는 일반화 잘 되는 하위문제를 맡고, 단면적은 연속방정식이 해석적으로 제공. PINN이 두 물리를 함께 강제.

## 3. 물리적 장치 (기준 형상) — 자동화 검증/교정 리그

- 링: 두께 **15mm**, **최대 10개** 적층. 링 스택 높이 = (링 수)×15mm, 최대 **150mm** (가변) = **추정 대상 라벨 높이**.
- 링 내경 6단계: 2cm~8cm → 반지름 **{10, 16, 22, 28, 34, 40} mm**.
- **상단 확장(고정)**: 링+피스톤이 5mm 하강, 100φ 아크릴 원통은 고정 → 최상단 링 위 **r=50mm·높이 5mm** 구간이 **입구 바로 안쪽에 항상 존재**. 이를 음향 공동에 **포함** → **음향 공동 높이 = 링스택 + 5mm, 최대 155mm**. (이 5mm는 추정 라벨이 아니라 순방향에 늘 들어가는 **기지 고정 세그먼트**.)
- 마이크: 무지향성, 컵 축 중앙, **음향 공동 상단(입구)에서 10mm 위** (= 바닥 기준, 10링이면 165mm). 평균 관구효과 기준 압력 마디 근처 → 공명에서 **딥(dip)** 측정.
- **장치의 역할**: 실험 **자동화** 시연 + 정답 아는 형상으로 방법 **검증** + 실제 마이크/스피커/공간 **시그니처 수집**(도메인 적응용). 이산 스텝은 과도기 단계.

- ❌ **금지**: "부피비율 등분(각 컵 자기 부피의 N등분)" — 절대 스케일/연속방정식 채널을 정규화로 파괴.

## 5. 순방향 시뮬레이션 (Forward)

- 1D 전달행렬법(TMM), **미분가능**하게 유지(향후 PINN 재사용). 폐-개, 물표면=강체.
- **마이크 응답 = B 정식 모델**: (공동 입구 위 **10mm**) 마이크 점 압력 = ① 스피커 직접장 + ② 입구 단극 재방사(`U_mouth ∝ p_입사/Z_mouth(f)`) 합. 공명에서 `Z_mouth` 최소 → 상쇄간섭 → **딥**. 상단 5mm 확장 반영.
- **감쇠 (✅ 결정 2026-07-02)**: **Kirchhoff 경계층 손실 공식으로 확정** — 점성·열 손실, 상수까지 알려진 표준 물리라 튜닝 파라미터 없음. 구현 = 파수를 복소수로(`k → (ω/c)(1-jα)`, `α ∝ √f/(r·c)`), TMM 구조·미분가능성 유지. 무손실 TMM의 면도날 피크/500빈 샘플링 문제도 함께 해결. (대기 흡음은 경계층 대비 미미 → 생략 가능.)
- **관측량 (✅ 결정 2026-07-02)**: 시뮬-실측 공통 기초 = **컵 없음 기준 측정** — 피스톤 최상단 = 공동 깊이 0 = 평평한 강체면. 자동화 사이클에 포함해 매 세션 확보. 이 기준으로:
  - **주 전략**: 절대 스펙트럼 + **시그니처 곱하기 증강**(§9) 학습 → 실녹음 직접 추론. 전날 현장 시그니처로 미세조정.
  - **안전망**: **비율 전달함수** `H_cup(f)/H_ref(f)` — 스피커·마이크·룸 특성(S·R·M)이 약분되어 시뮬이 직접 계산 가능한 순수 물리량. 현장 재학습 불필요. (한계: 덧셈성 소음은 약분 안 됨 → 잔여물 주입 증강 필요.)
  - 두 방식은 한 파이프라인의 증강 옵션 두 개로 구현(별도 구현 금지).
- **유효대역 (✅ 결정: 1D 유지)**: on-axis 대칭 셋업에선 각 모드 (1,0)이 축상 마디라 가진·감지 안 됨 → 실질 한계는 첫 반경 모드 `≈ 0.61·c/a`. 수치 검증 결과 r≤4cm에서 첫 반경 모드 ≥ **5,259Hz(대역 밖)** → **50~5000Hz에서 1D = 2D. 1D 유지.** 근거: `docs/analysis/1d_vs_2d_validband.{py,md,png}`. 재검토: 대역>5kHz 또는 비원형 단면.
- **생성 대역 (✅ 결정 2026-07-02)**: 시뮬 생성 **50~7000Hz, 700빈**(저역 빈 폭 ≈10Hz 유지). r 상한 50mm 확장으로 첫 반경 모드가 넓은 컵에서 대역 내 진입: r=50→4.2kHz, r=40→5.2kHz, r=35→6.0kHz, r=30→7.0kHz. **4.2kHz 이하는 전 형상에서 1D 유효.** 그 위 대응: (i) 학습 사용 대역 = **crop 하이퍼파라미터**(기본 5kHz), 리그 최광폭 링 실측 어블레이션으로 확정. (ii) 4~7kHz **가짜 딥 랜덤 주입**(도메인 랜덤화, §9)으로 미모델 반경 모드에 대한 과신 방지. 고차 모드의 가치: 축방향 해상도(λ/4 ≈ 12~17mm) + 거의 찬 상태(짧은 공기기둥 12~21mm)의 유일 공명이 4~7kHz에 존재.

## 6. 전처리 (✅ 2026-07-02 확정)

1. **스펙트럼 = log → z-score, "시퀀스 전체" 단위** — 게인은 세션 상수이고 곱셈이므로, log 후 평균 제거로 정확히 소거됨. **스텝별 개별 표준화 금지**: 스텝 간 진폭 관계(딥 깊이 변화 등)는 물리 신호라 보존해야 함. 평균 1개·표준편차 1개/샘플.
2. **V_i = 고정 상수(1L)로 정규화** — 샘플별 최대부피로 나누기 **금지**(절대 스케일 파괴 = §4 금지사항 "부피비율 등분"과 동일 오류).
3. **라벨 = 물리 상수 min-max**: r ∈ [5, 50]mm, H ∈ [30, 250]mm (§4 생성 범위와 동일 상수, 설정 파일에 고정). **정규화 기준을 데이터 통계에서 뽑지 말 것** — 구 코드의 spec_min/max 방식은 데이터셋 재생성 때마다 기준이 흔들려 모델-데이터 불일치 사고의 원인.
4. **가변 길이 = 패딩 + 마스크** — 무효 스텝/슬롯을 0 값으로 때우지 않고 0/1 마스크로 명시(모델 입력·손실 계산 양쪽에 적용).
5. 주파수 대역: §5의 crop 하이퍼파라미터를 따름.

## 7. 입출력 규격 (CNN/RNN/PINN 공통) — ★ 못박을 것 ★

- **입력**:
  - 스펙트럼 시퀀스 (가변 길이 → **마스킹**).
  - 스텝별 부은 부피 `V_i` (연속방정식 채널, 정당한 측정 입력).
- **출력 (✅ 2026-07-02 개정)**:
  - 반지름 **25슬롯 그리드**(피치 10mm, 절대 높이 0–10, …, 240–250mm) + **높이 H**.
  - 그리드 = **출력 샘플링 해상도**(≈물리 한계 λ/4)일 뿐 형상 가정 아님. **장치 링 두께 15mm와 비종속.** 라벨 = 각 밴드 **평균 단면적의 등가반지름** `√(Ā/π)` (연속방정식이 측정하는 양과 동일 형태).
  - 유효 슬롯만 손실 계산(**마스킹**). 연속 회귀 유지(분류 아님).
  - 리그 평가: 그리드-링 정렬 불필요. 예측 r(z) vs 정답 r(z) **함수 대 함수 비교**.
- **역할 분담**: 대략 스케일/높이 ← 유체(V), 정밀 형상(10반지름) ← 음향.

## 8. 모델 사다리 (CNN → RNN → PINN)

각 단계 정확도 향상 = 기여의 증거. **세 단계가 7장 규격을 공유**해야 비교가 깨끗함.

- **CNN**: 20×500을 2D 패턴으로. 베이스라인.
- **RNN(LSTM)**: 물채움을 시계열로. 어블레이션으로 "순서가 기여하나" 검증
  (셔플 테스트 / 역방향·양방향 / 마지막-스텝-only vs 전체 풀링·어텐션 / **구간별(상·하부) 오차**).
  - 물리 예측: 바닥 형상은 초기(긴 기둥) 스텝에서만 보임 → 역방향/양방향은 **바닥 반지름 오차**를 특히 줄여야 함.
- **PINN**: 미분가능 TMM 물리층. 손실 =
  1. 음향 일치: `TMM(공기기둥[h(t),H], A) ≈ 측정 스펙트럼`
  2. 연속방정식 잔차: `∫A dh = Q·t`
  3. 수면 위(음향)·아래(연속) A 겹침 일치.

## 9. 검증 설계 + 도메인 적응 전략 (✅ 2026-07-02 확정)

- 도메인 갭 분리: **시뮬 계단컵** 정확도 vs **실측 계단컵** 정확도 → 순수 sim→real 갭.

### 9.1 시그니처 = 2성분 분리 수집
- **곱셈 성분** (스피커·마이크·룸 전달특성 S·R·M): 각 공간에서 **스윕 측정 + 기준형상(피스톤 최상단)으로 나눠 추출**. cavity 응답은 제외.
- **덧셈 성분** (배경소음): 각 공간에서 **그냥 녹음** (복도 백색소음, 관중 웅성거림 등).
- 라이브러리: 실험실/강당/밴드실/복도 등 여러 공간에서 두 성분을 각각 채취. 곱셈 성분의 가상 확장은 pyroomacoustics 등 룸 시뮬레이션으로(스피커로 공간음 재생·녹음하는 방식은 덧셈 성분에만 유효 — 부록 A).

### 9.2 on-the-fly 증강 (데이터에 굽지 않고 로더에서 실시간)
- 매 샘플마다 랜덤 조합: **(룸 시그니처 1개 곱하기) + (소음 1개 더하기) + (섭동: 게인·스펙트럼 기울기·미세 시프트·온도 15~30°C·마이크 높이 d_mic ±1mm)**.
  - d_mic 지터 근거: 딥 위치의 근접장 이동 민감도 ≈ 1.7Hz/mm(모드1, 빈 폭 10Hz 대비 소폭) → 지그 공차 흡수용.
- **스텝 서브샘플링 증강 (✅ 2026-07-02)**: 저장된 풀 시퀀스(10mL 격자)에서 로더가 랜덤 부분수열을 추출해 학습(V_i는 누적값 그대로). 효과: ① 가변 간격 V_i 패턴이 학습분포 안으로 → **적응형 프로토콜 v2(10mL 배수 양자화) 추론을 v1 데이터셋 하나로 커버**(재생성 불필요), ② 스텝 번호에 기대는 지름길 차단 → V_i(유체 채널)를 실제로 읽도록 강제.
- 라이브러리 몇 개 × 조합 = 수백 가상 환경. 시그니처 파일 교체만으로 재증강 가능(전날 밤 작업 최소화).

### 9.3 학습 전략
- **베이스 모델 1개**: 전 라이브러리 혼합(멀티컨디션) 학습. 환경별 전용 모델보다 미지 환경 일반화 우수(부록 A).
- **현장 미세조정**: 전람회 전날 설치 → 현장 시그니처·소음 채취(자동) → 베이스에서 fine-tuning(낮은 LR, 짧게, 현장:라이브러리 혼합 유지로 망각 방지).
- **안전망**: 비율 방식(§5) 사전 학습 모델 지참 — 현장 재학습 실패 시 대체.
- **어블레이션(발표 자료 겸용)**: ① 환경별 전용 vs 혼합 베이스, ② 절대+시그니처 vs 비율, ③ 미세조정 전 vs 후.

### 부록 A. 지난 논의에서 교정된 오해(기록용)
- "부피비율 등분 샘플링" → **금지**(연속방정식 채널 파괴). 정유량+고정 ΔV로.
- "부은 부피 입력 = 정답 누설" → **아님.** V는 유체 측정 채널(정당). 형상은 여전히 음향에서.
- "높이/L_air 입력" → 금지(정답 파생). **V(부은 부피)만** 입력.
- "하수구 유량계가 메인" → 아님. **음향 형상측정의 유일성 복원**이 핵심 명제.
- "1D는 2500Hz까지만 유효" → 과장. on-axis라 실질 한계는 첫 반경 모드 ≈0.61c/a. **50~5000Hz에서 1D 유효(수치 검증, `docs/analysis/1d_vs_2d_validband`)** → 1D 유지, 2D 불필요.
- "2D 축대칭 → 원/사각 구별" → 아님. 축대칭은 원형을 가정. 단면 모양 구별은 각 모드(m≥1)=완전 3D 필요(+on-axis 마디+대역 밖으로 삼중 장벽).
- "스피커로 유튜브 공간음을 재생해 룸 시그니처 생성" → **덧셈 성분(소음)으로만 유효.** 곱셈 성분은 재생해도 실제 방+재생 스피커의 응답이 찍힘. 가상 룸 전달함수는 pyroomacoustics 등 물리 시뮬레이션으로.
- "부피 등분 + ΔV를 입력으로" → 정보 파괴는 해소되나 **정답 누설(label leakage)**: ΔV=V×0.95/N은 정답 파생값 → 모델이 부피를 소리 없이 역산하는 지름길 학습 → 실전 붕괴. 게다가 실전에서는 V를 몰라 등분 불가(닭-달걀).
- "지수 증분(10,20,40...)" → 누설·닭달걀은 없으나 압축 좌표가 틀림: 마지막 붓기가 컵의 절반 → **위쪽 해상도 붕괴** + 스냅샷 수 급감. 올바른 좌표는 높이 → **등-Δh 적응형**(§4 v2)이 지배해.
- "환경별 전용 모델을 만들어 상황에 맞게 선택" → 혼합(멀티컨디션) 학습 1개 + 현장 미세조정이 일반적으로 우수(전용 모델은 시그니처 과적합 + 환경 판단이 추가 오류원). 전용 vs 혼합은 어블레이션으로 남김.
- "출력 슬롯 = 장치 링 두께 15mm" → 아님. 출력 그리드는 답안의 **샘플링 해상도**로, 물리(λ/4 ≈ 12~17mm + 연속방정식 채널)가 정하는 설계값(10mm 채택). 형상 생성은 범위 내 완전 연속·랜덤이며 장치와 비종속.
